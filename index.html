Personal Work

Sample map created for Seattle Utilities depicting the different pipe repairs downtown

Using Seattle Open Data this map was created posing as a member of the Seattle Public Utilities GIS team. The goal was to showcase the different repairs to the pipe infastrucutre that has happened in the downtown area.

University of Washington Football Recruits

Tutorial created by John Nelson was the inspiration for creation of this map.

This map included the use of ESRI's geocoding data base. After collecting and generating an excel document depicting the recruiting classes hometowns was able to use the geocoder to cretae a list of latitude and longitude points for each player. Then created lines connecting both the Universtiy of Washington to each of their home towns to create a pleasing Visualization of where each memeber of the recruiting class traveled from to University of Washington.
